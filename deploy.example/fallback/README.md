# Кастомные fallback-заглушки

Этот каталог — часть шаблона **`deploy.example/`**. После копирования в рабочий каталог путь будет **`deploy/fallback/`** (рядом с `deploy/inventory` и `deploy/secrets.enc`).

Положите сюда `index.html` и/или `403.html`, чтобы заменить стандартные заглушки HAProxy.

- **index.html** — показывается при запросе `/` (маскировка «Site Under Construction»)
- **403.html** — показывается при 403 Forbidden

Файлы из **`deploy/fallback/`** накладываются поверх дефолтных из `fallback/` (корень репозитория с playbook’ами).
Можно переопределить только один файл — второй останется стандартным.
